The Tale of Gopi and Shar Khan

Addendum

Planning & Design Choices

  • Started with a flowchart approach to map out all possible story paths and ensure each decision led to meaningful consequences
  • Chose a nature-inspired color scheme (greens and earth tones) to match the story's setting
  • Designed the layout to be clean and focused, with the story text and choices clearly separated

Technical Implementation

  • Used object-oriented data structure for story management, making it easy to modify and extend
  • Implemented responsive design principles to ensure the game works well on all devices
  • Created reusable functions for game state management and scene transitions

Challenges & Solutions

  • Challenge: Managing multiple story paths without repetitive code
    Solution: Implemented a centralized story data object with clear node connections
  • Challenge: Maintaining consistent image sizing across different devices
    Solution: Used percentage-based dimensions with centered positioning
  • Challenge: Ensuring smooth transitions between scenes
    Solution: Implemented state management system with clean update functions

Insights Gained

  • Learned the importance of proper story structure planning before coding
  • Discovered effective ways to manage complex state in interactive narratives
  • Gained experience in creating engaging user interfaces for storytelling
