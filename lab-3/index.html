Lab 3: Serving fonts

For this assignment, I have choosen google fonts to serve the fonts, because it have lots of fonts to choose, and it is easy to use. So the process is simple, just go to the google fonts website, choose the font you want, and then copy the link tag and paste it in the head section of your html file. Then you can use the font in your css file. We can also use fonts which are already installed in our system, but it is not recommended because the font may not be available in all the systems. So it is better to use google fonts or any other font service to serve the fonts. The fonts I have choosen are Handlee, Amatic SC and Permanent Marker. I have also given the font size and font weight for each font. I have used Handlee for the heading, Amatic SC for the paragraph and Permanent Marker for the footer. I have also used javascript to display the last modified date of the document.
